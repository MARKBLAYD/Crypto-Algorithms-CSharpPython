# Криптографические стандарты и протоколы (C#)

Учебные проекты по реализации алгоритмов криптографической защиты информации и протоколов аутентификации. Все решения реализованы в виде консольных приложений на C# (.NET).

## Реализованные алгоритмы
*   **Лабораторная 1:** Алгоритм блочного шифрования «Магма» (ГОСТ Р 34.12-2015)
*   **Лабораторная 2:** Алгоритм RSA
*   **Лабораторная 3:** Хэш-функция ГОСТ 34.11-2012 («Стрибог»)
*   **Лабораторная 5:** Параллельная схема аутентификации Фиата-Шамира
*   **Лабораторная 6:** Протокол разделения секрета Шамира
*   **Лабораторная 7:** Слепая подпись Чаума
*   **Лабораторная 8:** Скрытый канал на основе схемы Онга-Шнорра-Шамира

## Как запустить
Для запуска необходим установленный [.NET SDK](https://dotnet.microsoft.com/download).

1. Перейди в папку с нужной лабораторной:
```bash
cd Lab1-Magma
```
2. Восстанови зависимости и скомпилируй проект:
```bash
dotnet build
```
3. Запусти программу, передав пути к входному и выходному файлам:
```bash
dotnet run -- C:\input.txt C:\output.txt
```
